HopeFront NGO Template Documentation

Version: 1.0.0
Author: Antigravity
Framework: Next.js 14 (App Router) + React 18

1. Introduction

Thank you for purchasing the HopeFront NGO Template. This template is designed specifically for charities, non-profits, and fundraising organizations. It is built with Next.js and Vanilla CSS modules to provide maximum performance, excellent SEO, and easy customizability without being locked into complex utility CSS frameworks.

2. Installation & Setup

Requirements: Ensure you have Node.js 18.x or later installed on your system.
  1. Extract the downloaded zip file to your preferred location.
  2. Open your terminal and navigate to the project directory.
  3. Install dependencies by running: 
    npm install
  4. Start the development server: 
    npm run dev
  5. Open http://localhost:3000 in your browser to see the template.

3. Project Structure

The project follows standard Next.js App Router conventions:

  • /app - Contains page layouts, global CSS, and main routes (e.g., page.tsx, layout.tsx).
  • /components - Contains all reusable React components (Navbar, Hero, CampaignCard, Footer) and their respective CSS modules.

4. Customization

Changing Colors and Fonts

All global styles and theme variables are managed in app/globals.css. To change the brand colors, locate the :root variables and update them:

:root {
  --primary-color: #YourColor;
  --secondary-color: #YourColor;
  --accent-color: #YourColor;
}

Fonts are loaded via Next.js Font Optimization in app/layout.tsx. You can easily swap 'Inter' and 'Outfit' with other Google Fonts.

5. Deployment

This Next.js application can be seamlessly deployed to Vercel, Netlify, or any standard Node.js hosting environment.

npm run build
npm run start
